** Connect to SAP IQ

      --database=<database>
                          Database name
      --host=<host>       Host name
                          Optional, defaults to localhost
      --port=<port>       Port number
                          Optional, defaults to 50000
      --server=<server>   --server=sqpiq
                          Loads SchemaCrawler plug-in for SAP IQ
Add connection options to the `connect` command in the SchemaCrawler Shell
